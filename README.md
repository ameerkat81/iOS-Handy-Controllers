## Classes & Controllers description:
### HudView
HudView is a custom UIView. Usage example: after hit save button, show this view to give user a heads-up display.
HudView 是一个自定义的UIView。使用情境：点击保存按钮之后，向用户展示此View来提醒用户是否保存成功
